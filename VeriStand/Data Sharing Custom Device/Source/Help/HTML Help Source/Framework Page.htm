Data Sharing Framework

The Data Sharing Framework is an add-on that provides a plugin-based approach to reflecting data between applications.

What do you want to do?

Create a new Data Sharing Framework configuration

Click the New button to launch the configuration window and create a new configuration for the Data Sharing Framework.

Import an existing Data Sharing Framework configuration

Click the Import button to import an existing configuration for the Data Sharing Framework. These must be valid JSON files of the extension .dsf. This option will override your exiting VeriStand channel configuration.

Export the current Data Sharing Framework configuration

Click the Export button to export the current configuration for the Data Sharing Framework. This will reflect any changes to settings made inside the custom device tree.
